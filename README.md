# PizaaSchool
=====
피자가 액션 점프를 하면서 오래 살아남는 게임 프로그램입니다.
한번 쯤은 게임을 만들어 보고 싶다고 생각했었는데 게임에 대한 게임방식을 분석하고, 프로그램으로 구현을 하면서 추가적으로 아이템을 추가하고 스테이지를 구현하였습니다.
